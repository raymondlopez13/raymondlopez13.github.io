Now Playing

Now Playing is a social media application that uses SQL, Express, and Handlebars so users can post albums that they are currently listening to. Users can create an account and customize their profile as well as makes posts. There is also a search feature so users can find their friends and view their posts, also users can filter posts to look for certain genres, media type, and release date.

Cinequiz

Cinequiz is an interactive front end game that uses Giphy API and OMDb API to test users on their cinema knowledge. Each round, the user is given the name of a movie and six actors. Fourof the actors are in the movie and two are not. Choose correctly and you earn a point, choose incorrectly and you lose a point and are taken to the next movie. If you are stumped, you have the option to move onto the next question without worrying about choosing the wrong actor. High scores are kept in local storage and can be cleared when the user chooses to do so.
